# Unoclave

Unoclave is an AI-powered platform designed to assist users with queries related to household appliances. Whether it's troubleshooting issues with refrigerators, washing machines, or other home devices, Unoclave's intelligent chatbot provides efficient and reliable solutions tailored to user needs.

## Key Features

- **AI-Powered Chatbot**: Ask queries about any household appliance, and the chatbot offers instant solutions.
- **Category-Based Assistance**: Solutions are tailored based on appliance type, brand, and model number (optional).
- **User-Friendly Interface**: Easy-to-use chat functionality ensures a smooth user experience.

## Tech Stack

### Frontend

- **Next.js**: Modern framework for building fast, scalable, and SEO-friendly web applications.

### Backend

- **Node.js**: A runtime environment for building high-performance applications.
- **Express.js**: A minimal and flexible web application framework for building the API.

## How It Works

1. **User Interaction**: Users interact with the chatbot by typing queries related to their home appliances.
2. **AI Assistance**: The chatbot processes the queries and provides relevant solutions in real-time.

3. **Query Categorization**: Solutions are tailored based on optional inputs like appliance category, brand, and model number.
